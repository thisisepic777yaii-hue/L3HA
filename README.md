
**To stop:** Press `Ctrl+C` in the command prompt

## How to Stop It

The program is designed to be completely stoppable:
1. Press `Ctrl+C` in the command prompt window
2. All effects will stop immediately
3. Your desktop returns to normal
4. No permanent changes are made
